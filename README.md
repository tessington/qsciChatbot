# qsciChatbot

**qsciChatbot** is a Shiny-based virtual coding assistant built with
`{golem}` and OpenAI to help students debug R code in quantitative
ecology courses.

## Installation

``` r
# Install from GitHub
devtools::install_github("tessington/qsciChatbot")

# load in R
library(qsciChatbot)
```
